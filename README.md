## 🌟 Giới thiệu

**Share Truyện** là một web app đọc và chia sẻ truyện chữ được xây dựng nhằm mang lại trải nghiệm đọc dễ chịu hơn so với các website truyện truyền thống.

Ứng dụng tập trung vào:

> giao diện sạch • tốc độ nhanh • tối ưu mobile • đọc liên tục không rối mắt.

Người dùng có thể dễ dàng:
- khám phá các bộ truyện mới,
- đọc chương mượt mà,
- tìm kiếm nhanh theo tên truyện,
- và chia sẻ nội dung với cộng đồng.

---

## ✨ Tính năng chính

- 🎨 Giao diện hiện đại, hỗ trợ Dark / Light Mode
- 📖 Trình đọc truyện tối ưu cho đọc lâu
- ⚡ Tốc độ tải nhanh với Vite + TypeScript
- 📱 Responsive hoàn toàn trên mobile/tablet
- 🔍 Tìm kiếm truyện nhanh chóng
- 🗂️ Quản lý chương truyện và danh mục
- 🔗 Chia sẻ link truyện dễ dàng

## 🛠 Công nghệ sử dụng

- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Build tool**: Vite
- **Ngôn ngữ**: TypeScript


## 🚀 Cách chạy dự án locally

### 1. Clone repo
```bash
git clone https://github.com/ineqien0302-a11y/share-truyen.git
cd share-truyen
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start development server

```bash
npm run dev
```
